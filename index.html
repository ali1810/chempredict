Architecture

Three ML Models

Each model trained on curated datasets from KIT PhD research.

Solubility · Model 1
XGBoost · 298 Features
Fast, interpretable gradient boosting on tabular molecular descriptors.
125 RDKit molecular descriptors
128-bit Morgan fingerprints
7 functional group flags
38 structural features
MW < 600 domain check
Solubility · Model 2
MPNN · Molecular Graph
Message-passing NN on molecular graphs without handcrafted features.
51 atom node features
6 bond edge features
4 message-passing layers
128-dim node embeddings
Mean + max readout → MLP
Retrosynthesis
Transformer Seq2Seq
Product SMILES → reactant SMILES via neural machine translation.
USPTO-50K · 10 reaction classes
16× SMILES augmentation
256-dim · 8 heads · 6 layers
Top-1: 61.3% · Top-5: 74.2%
Canonicalisation deduplication
Pipeline
XGBoost: SMILES → RDKit (125 desc + 128 Morgan + 7 FG + 38 struct) → 298 features → logS MPNN: SMILES → graph (atoms=nodes, bonds=edges) → 4× MP layers → readout → logS Retrosyn: Product SMILES + <RX_N> token → Transformer encoder → beam search → top-k reactants
